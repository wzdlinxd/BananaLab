### 作业

#### 目标：
1、实现 StringInterface 接口，实现一个容器为数组的 ArrayString。

2、实现 StringInterface 接口，实现一个容器为链表的 LinedString。

3、实现字符串匹配算法 indexOf。

4、实现数字转字符串的算法。

5、实现字符串首尾翻转字符串算法。

6、描述为什么在 JDK 中使用数组而不是链表来实现字符串。

7、描述为什么 String 是一个不可变的类，以及是怎么实现不可变的。
